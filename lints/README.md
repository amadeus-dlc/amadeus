# Lints

このディレクトリは、Amadeus 固有の lint を置く場所である。

`lints/check.ts` は `lints/*/check.ts` を検出して実行する。各 lint は、原則として `--check` と `--report` を提供する。閾値のように判定基準そのものを表すオプションは許容するが、違反を許容リストへ逃がすオプションは持たない。

lint 違反が出た場合は、対象コードを修正する。lint の判定基準が不適切な場合は、例外リストではなく lint 仕様を見直す。

## public-type-file

1 ファイルに複数の公開型を置かないことを検査する。公開型を支える非公開実装は、同じファイルに置いてよい。

## ts-complexity

TypeScript 関数の循環的複雑度を検査する。閾値を超えた場合は、条件分岐の整理や関数の分割によって解消する。

# ドメインモデル

## 範囲

このモデルは、BC001 認証アクセスの全体概念関係を扱う。

サブドメインと境界づけられたコンテキストの一覧は、[../../subdomains.md](../../subdomains.md) と [../../bounded-contexts.md](../../bounded-contexts.md) を参照する。

## 概念間の関係

- 利用者はアカウントに紐づく。
- アカウントは認証情報を持つ。
- アカウントはパスワード再設定を要求できる。
- パスワード再設定は再設定トークンを発行する。
- 再設定トークンはアカウントと有効期限に紐づく。

## 契約

不変条件は、[contracts.md](contracts.md) を参照する。

## ライフサイクル

```
パスワード再設定要求済み
  -> 再設定トークン発行済み
  -> 再設定トークン検証済み
  -> 認証情報更新済み
  -> 再設定トークン消費済み
```

## 集約候補

### アカウント

アカウントは認証情報とパスワード再設定の整合性を守る候補である。

再設定トークンの発行、検証、消費はアカウントに紐づく操作として扱う。

# B001：システムがパスワード再設定要求の流れを提供する

## 対象ユニット

- U001

## 要求

- R001

## ストーリー

- S001

## ユースケース

- UC001

## 範囲

パスワード再設定要求を受け付け、再設定トークンを発行する。

## 設計

- [U001 Unit Design](../../units/U001-password-reset-request/design.md)

## 完了条件

- R001 の受け入れ条件を満たす証拠が揃う。
- 再設定トークンが期限付きで発行される。
- アカウントの存在有無を推測できない応答を維持する。

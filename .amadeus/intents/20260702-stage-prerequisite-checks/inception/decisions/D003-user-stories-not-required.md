# D003 User Stories 省略

## 状態

accepted

## 文脈

この Intent の主な相互作用主体は、phase skill 起動時に判断する Agent である。

Maintainer は stage0 採用判断やレビューで関与するが、要求の中心は人間アクターの価値表現ではなく、phase skill 起動時の前提確認である。

## 判断

User Stories は作らない。

ユースケースの Story 参照は `なし` にする。

## 影響

`state.json.inception.requiredStoryArtifacts` は空配列にする。

追跡では、ユーザーストーリー列に `なし` を記録する。

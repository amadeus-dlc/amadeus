# D002 README の内部 skill family

## 状態

accepted

## 判断

README の内部 skill は、個別の公開入口として並べず、workflow family ごとに整理する。

## 根拠

内部 skill は Amadeus workflow の中で使われる。

README で内部 step を列挙する場合でも、公開入口である phase skill と混同しない表現が必要である。

## 影響

README.md と README.ja.md に、内部 skill の family 表を追加する。

既存の公開入口説明は維持する。

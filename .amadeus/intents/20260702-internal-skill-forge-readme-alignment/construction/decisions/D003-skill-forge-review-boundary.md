# D003 skill-forge 確認境界

## 状態

accepted

## 判断

Amadeus skill を確認または変更するときは、`skill-forge` で skill 境界、trigger description、本文指示、eval coverage、Codex metadata を確認する。

## 根拠

`skill-forge` は skill 自体の作成、改善、確認を扱う補助 skill である。

Amadeus skill の変更時に確認観点を README に置くことで、内部 skill の役割整理と互換性確認を同じ入口から参照できる。

## 影響

README.md と README.ja.md に確認観点を追加する。

`skill-forge` の SKILL.md は変更しない。

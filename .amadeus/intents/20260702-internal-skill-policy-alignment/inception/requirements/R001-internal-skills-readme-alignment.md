# R001: Internal Skills 一覧照合

## 要求

`README.md` と `README.ja.md` を含む `README*.md` の Internal Skills 一覧が、現在の内部 skill 構成と照合されている。

## 根拠

- [Issue #284](https://github.com/amadeus-dlc/amadeus/issues/284)
- [scope.md](../../ideation/scope.md) の SC-IN-001
- [codebase-analysis.md](../codebase-analysis.md)

## 受け入れ状態

- `README*.md` の Internal Skills 一覧に、追加対象の内部 skill が反映されている。
- README に載せる対象と載せない対象の判断を、Inception または Construction の成果物から追跡できる。
- README の英語版と日本語版で Internal Skills の対象が対応している。

## 対象外

- `SKILL.md` 本文の英語化。
- 対象外候補の詳細判断。

# UC002 承認待ちがないことを確認する

## ユースケース

Maintainer が一覧を実行し、承認待ちが 0 件であることを示す表示から、承認の詰まりがないと判断する。

## アクター

- ACT001 Maintainer

## 外部システム

- なし

## 事前条件

- workspace のすべての Intent が承認待ちの状態を持たない。

## 基本フロー

1. Maintainer は workspace を指定して一覧を実行する。
2. 一覧は `.amadeus/intents/*/state.json` を横断スキャンし、承認待ちを 1 件も検出しない。
3. 一覧は承認待ちが 0 件である旨を表示し、正常実行として終了する。
4. Maintainer は承認の詰まりがないと判断し、他の作業へ戻る。

## 代替フロー

| 条件 | 扱い |
|---|---|
| Intent が 1 件も存在しない workspace を指定した。 | 承認待ち 0 件と同様に、その旨が分かる表示で正常終了する。 |

# R003 承認待ち 0 件時の表示

## 要求

承認待ちが 0 件の場合も、出力からその旨が分かる。

## 背景

承認待ちがないことは「詰まりがない」ことの確認であり、空出力やエラーでは 0 件と実行失敗を区別できない。

## 受け入れ条件

- 承認待ちが 0 件の workspace で実行すると、承認待ちがないことを示す表示が得られる。
- 0 件は正常実行として扱われ、実行失敗と区別できる。

## 依存

R002。

## 対応する対象境界

- SC-IN-002

## 未確認事項

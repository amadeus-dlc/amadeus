# R003 生成マーカー

## 要求

再生成された `intents.md` と `discoveries.md` は、ファイル先頭に生成物であることと編集先（配下モジュール）を示すマーカーを持つ。

## 背景

インデックスが生成物になった後、ファイル単体を見た人間やエージェントが生成物と気づかずに直接編集する事故を防ぐ必要がある。
grilling の確定判断 GD003 により、全体生成 + 先頭マーカーの方式を採用する。

## 受け入れ条件

- 生成されるインデックスの先頭に、生成物であることと編集先を示すマーカーが含まれる。
- マーカーは Markdown の表示を壊さない形式である。

# UC001 wave を導出する

## ユースケース

Agent が `bolts.md` の依存表から wave（並行実行できる Bolt の集合）を導出し、実行計画として人間に提示する。

## アクター

- ACT002 Agent

## 外部システム

- なし

## 事前条件

- 対象 Intent の `bolts.md` が `依存` 列と `依存関係` 表を持ち、validator の構造検査を通る。

## 基本フロー

1. Agent は `bolts.md` の依存表を読み、各 Bolt の依存を確認する。
2. 依存がすべて前の wave までに完了する Bolt の集合として、wave 1 から順に wave を導出する。
3. 導出した wave 分割（wave 番号、Bolt、並行可否）を実行計画として人間に提示する。

## 代替フロー

| 条件 | 扱い |
|---|---|
| 依存に循環がある。 | wave を導出せず、`bolts.md` の依存表の補修へ戻す。 |
| すべての Bolt が直列依存である。 | 各 wave が 1 Bolt になり、従来どおりの直列実行と同じ計画になる。 |

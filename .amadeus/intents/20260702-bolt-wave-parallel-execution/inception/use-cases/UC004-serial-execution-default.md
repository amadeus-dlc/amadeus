# UC004 従来どおり直列で実行する

## ユースケース

Agent が、wave 並行の条件を満たさない場合（単一 worktree での作業、依存のない Bolt が複数ない場合）に、従来どおり Bolt を直列に実行する。

## アクター

- ACT002 Agent

## 外部システム

- なし

## 事前条件

- 対象 Intent の Construction を単一 worktree で進めている、または wave 内の Bolt が 1 件である。

## 基本フロー

1. Agent は wave 並行の条件（依存のない Bolt が複数あり、worktree 分離で実行できる）を確認する。
2. 条件を満たさない場合、従来どおり Bolt を 1 件ずつ、準備、承認、実装、検証の順で直列に実行する。
3. 内部プロセスの順序と Task Generation Gate の契約は、wave 並行の場合と同じに保たれる。

## 代替フロー

| 条件 | 扱い |
|---|---|
| 途中から worktree を分離できるようになった。 | 次の wave から並行実行へ切り替えられる。 |

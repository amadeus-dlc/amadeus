# R001 wave 導出の契約

## 要求

bolts の依存グラフから wave（並行実行できる Bolt の集合）を導出する契約が Construction skill から読め、同じ依存表からは常に同じ wave が導出される。

## 背景

`bolts.md` の `依存` 列と `依存関係` 表は必須構造として確定しており、依存グラフは機械的に読める。
wave は「依存がすべて前の wave までに完了した Bolt の集合」（トポロジカルレベル）として決定論的に導出できる。
#334 では B002 と B003 が設計上 B001 の後に並行実行可能だったが、導出の契約がなく直列実行された。

## 受け入れ条件

- wave の導出規則（依存がすべて前の wave までに完了した Bolt の集合）が Construction skill から読める。
- 同じ `bolts.md` の依存表からは常に同じ wave 分割が導出される。
- 依存に循環がある場合の扱い（導出せず bolts.md の補修へ戻す）が読める。

## 依存

なし。

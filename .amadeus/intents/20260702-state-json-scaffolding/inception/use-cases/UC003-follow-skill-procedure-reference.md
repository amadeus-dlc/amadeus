# UC003 skill 手順の参照に従ってスクリプトを使う

## ユースケース

Agent が phase skill の state 更新手順を読み、手書きの代わりに参照されたスクリプトで雛形を生成、更新する。

## アクター

- ACT002 Agent

## 外部システム

- なし

## 事前条件

- 対象 phase skill の手順に、スクリプトの利用が参照として書かれている。
- source skill と昇格先成果物が promote 手順で同期されている。

## 基本フロー

1. Agent は、phase skill の手順に従って作業を進め、state 更新の手順に到達する。
2. Agent は、手順に参照されたスクリプトを遷移種別を指定して実行する。
3. Agent は、先例の手書き再現を行わず、生成結果を成果物に含める。

## 代替フロー

| 条件 | 扱い |
|---|---|
| 手順に参照がない skill で state 更新が必要になる。 | 参照追加の不足として報告し、従来どおり手書きで補いつつ後続で手順を補修する。 |

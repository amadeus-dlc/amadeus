# ペルソナ — metrics-observation

| ペルソナ | 説明 | 本 intent での接点 |
|---|---|---|
| **P1: 品質トレンドの観測者** | leader/ユーザー — リポジトリの健全性推移を定点で見たい(#921 の一次要望者) | metrics/ の JSON 群を読み時集計し、劣化傾向・改善効果を判断 |
| **P2: 変更の実装者(builder/メンバー)** | 自分の変更が品質指標に与えた影響を事後確認したい | マージ後 snapshot の commit フィールドで自変更との因果を特定 |
| **P3: CI/計測基盤の保守者** | snapshot 機構自体の保守(collector 追加・計測器更新) | スキーマの疎結合(FR-5)と tool_version 記録で保守作業が独立化 |

# D003：完了証拠

## 判断

子 Issue の完了は、対応 PR の merge または明示的な Issue close で観測する。

## 状態

active

## 根拠

実装 PR を伴う Issue と、方針判断やスコープ外判断で close する Issue の両方を扱えるようにするためである。

完了証拠が確認できない子 Issue は、完了扱いにしない。

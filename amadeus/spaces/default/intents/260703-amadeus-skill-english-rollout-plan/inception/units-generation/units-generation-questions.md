# Questions：Units Generation

## 回答一覧

| ID | 質問 | 推奨回答 | 回答 |
|---|---|---|---|
| Q001 | Unit の境界戦略と粒度をどうしますか。 | 機能別で中粒度。#395、#400、#401、#402、#391〜#394、RU002〜RU006 を基本単位にする。 | 推奨回答を採用する。 |

## Q001 の判断

Unit 境界は、子 Issue、差分対応、英語化 rollout unit を基本にした機能別、中粒度とする。

#401 配下の #391、#392、#393、#394 は、U003 で順序を確定し、U005 で個別完了または対象外判断を追跡する。

#399 の完了判断は、U010 の到達条件として扱う。

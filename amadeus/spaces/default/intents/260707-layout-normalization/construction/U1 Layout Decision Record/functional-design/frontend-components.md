# Frontend Components — U1 Layout Decision Record

## Applicability

U1 は UI を含まない。`unit-of-work`, `unit-of-work-story-map`, `requirements`, `components`, `component-methods`, `services` は repository documentation/design の作業を定義しており、frontend component hierarchy は存在しない。

## Non-UI Interaction

利用者の interaction は Pull Request review と docs/design record review である。UI component、props/state、form validation、API integration は対象外である。

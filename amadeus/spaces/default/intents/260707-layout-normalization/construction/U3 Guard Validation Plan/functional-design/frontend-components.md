# Frontend Components — U3 Guard Validation Plan

## Applicability

U3 は UI を含まない。`unit-of-work`, `unit-of-work-story-map`, `requirements`, `components`, `component-methods`, `services` は validation plan を定義しており、frontend component は存在しない。

## Non-UI Interaction

利用者の interaction は command execution と PR review である。UI hierarchy、props/state、form validation、API integration は対象外である。

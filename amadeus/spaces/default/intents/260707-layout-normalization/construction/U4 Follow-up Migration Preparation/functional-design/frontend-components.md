# Frontend Components — U4 Follow-Up Migration Preparation

## Applicability

U4 は UI を含まない。`unit-of-work`, `unit-of-work-story-map`, `requirements`, `components`, `component-methods`, `services` は follow-up planning を定義しており、frontend component は存在しない。

## Non-UI Interaction

利用者の interaction は issue/backlog/design record review である。UI hierarchy、props/state、form validation、API integration は対象外である。

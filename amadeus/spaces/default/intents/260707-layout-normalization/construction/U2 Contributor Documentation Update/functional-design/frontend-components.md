# Frontend Components — U2 Contributor Documentation Update

## Applicability

U2 は UI を含まない。`unit-of-work`, `unit-of-work-story-map`, `requirements`, `components`, `component-methods`, `services` は documentation update を定義しており、frontend component は存在しない。

## Non-UI Interaction

利用者は README/docs を読む contributor である。UI hierarchy、props/state、form validation、API integration は対象外である。

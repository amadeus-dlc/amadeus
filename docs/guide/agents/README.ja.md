# エージェント詳細解説

各エージェントごとに 1 ページを割き、その担当領域、リードおよびサポートするステージ、ロードする知識、そして協働の仕方を解説します。これらは[エージェントの章](../06-agents.md)のロスターを掘り下げたものです — なぜ 30 の狭いエージェントではなく 11 の広いエージェントなのかという設計思想はまずそちらを読み、個々のエージェントの詳細はここで読んでください。

技術リファレンス(フロントマター契約、ツール許可リスト、比較マトリクス)については、開発者リファレンスの[エージェントリファレンス](../../reference/agents/README.md)を参照してください。

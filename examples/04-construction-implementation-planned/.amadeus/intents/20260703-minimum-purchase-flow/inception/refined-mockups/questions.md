# Questions：Refined Mockups

人間の指示が「質問せずに続行し、判断が必要な点は指示と既存成果物から最も妥当な内容を選ぶ」と確認論点への回答方針を先に与えたため、grilling による対話は行っていない。
回答列には、指示と既存成果物から確定した内容を記録する。

| # | 確認したいこと | 推奨回答 | 回答 |
|---|---|---|---|
| 1 | 数量はどの画面で指定するか | 商品一覧の選択時に指定する | 商品一覧の各行に既定値 1 の数量入力を置き、選択時に確定する。注文内容の確認画面では変更しない。最小構成を優先した |
| 2 | 使うデザインシステムや UI 規約はあるか | 指定がなければ未確認として汎用部品分類へ対応付ける | 未確認である。Web UI フレームワークの確定後に具体部品へ対応付け直す前提で、汎用部品分類へ対応付けた |
| 3 | アクセシビリティで満たすべき基準は何か | WCAG 2.1 AA 相当を目安にする | 準拠基準の指定は未確認である。WCAG 2.1 AA 相当を目安に基本観点をチェックリスト化し、確定は Construction の NFR 要求に委ねる |

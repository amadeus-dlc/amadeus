# Frontend Components：注文作成

## 構成

| コンポーネント | 責務 | 対応するモック |
|---|---|---|
| 注文内容確認画面 | 注文内容（商品名、数量、価格、合計金額）の表示。注文を作成する操作と商品一覧へ戻る操作の受け付け。作成に失敗したことの表示 | [mockups.md](../../../inception/refined-mockups/mockups.md) の「注文内容の確認画面」 |
| 注文完了画面 | 注文の作成結果と注文番号の表示。商品一覧へ戻る操作の受け付け | [mockups.md](../../../inception/refined-mockups/mockups.md) の「注文完了画面」 |

作成に失敗したことの表示はモックがなく、文言は未確認である（interaction-spec.md の「注文内容の確認」の例外に対応する）。

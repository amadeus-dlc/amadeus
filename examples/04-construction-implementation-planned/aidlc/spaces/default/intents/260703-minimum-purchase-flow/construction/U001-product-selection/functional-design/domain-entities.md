# Domain Entities：商品選択

## 目的

商品選択が扱うドメインの概念と関係を定義する。

## Domain Entity

| Entity | 責務 | 主な属性 |
|---|---|---|
| 商品 | 注文対象の商品を表す | 商品識別子、商品名、価格 |

## 関係

商品は、商品一覧提供が返す一覧の要素であり、商品一覧画面の選択操作の対象である。
選択した商品の商品識別子は、注文作成（U003）の注文の入力になる。
在庫参照（U002）は、B002 で商品識別子を使って商品ごとの在庫状況を対応づける。

## Domain Map と Context Map への反映候補

| 候補 | 種別 | 判断 |
|---|---|---|
| 商品選択の語彙（商品） | 共有境界 | 見送り |

見送りの理由は次である。
Domain Map に採用済みの Subdomain と Bounded Context がなく、「商品」は本 Intent 内の領域間の受け渡しに使う語彙に留まるため、共有境界として採用する判断材料がない。

## 未確認事項

- 注文対象になる商品情報の出どころ（登録と更新の手段）は未確認である。

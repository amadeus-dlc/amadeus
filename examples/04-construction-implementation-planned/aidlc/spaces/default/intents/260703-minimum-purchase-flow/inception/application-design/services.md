# Services：260703-minimum-purchase-flow

## 一覧

| サービス | 責務 | 含むコンポーネント |
|---|---|---|
| 商品サービス | 商品一覧と在庫状況をまとめ、購入者が選択できる商品の情報として提供する | 商品一覧提供、在庫参照 |
| 注文サービス | 注文の作成と記録を行い、注文の識別子を返す | 注文作成、注文記録 |

サービスの分割は、ドメインの領域（商品選択、注文作成、在庫参照）に合わせる。
在庫参照は商品サービスに含める。最小購入フローでは在庫状況の利用先が商品一覧の表示だけであり、独立したサービスにする理由がないためである。

# Questions：Refined Mockups

| # | 確認したいこと | 推奨回答 | 回答 |
|---|---|---|---|
| 1 | 詳細化で確定すべき相互作用はどれか | 在庫参照失敗と在庫不足の例外系 | 在庫参照失敗（R008）と在庫不足（R007）の例外系を各画面の表示と相互作用として確定した。正常系は rough mockups の 3 画面を維持した |
| 2 | 使うデザインシステムや UI 規約はあるか | なし（汎用 UI 部品の語彙で対応づける） | 確立済みのデザインシステムはない。汎用 UI 部品の語彙で対応づけ、ライブラリ選定は Construction の技術スタック判断で扱う |
| 3 | アクセシビリティで満たすべき基準は何か | WCAG 2.1 レベル AA を目安にする | 明示された基準がないため WCAG 2.1 レベル AA を目安に基本観点を確認した。正式な基準の確定は未確認のまま Construction の NFR Requirements へ引き継ぐ |

本ステージは人間への逐次質問を行わず、実行指示と既存成果物から回答を確定した。

# Component Dependency：260703-minimum-purchase-flow

## 依存関係

| 依存元 | 依存先 | 理由 |
|---|---|---|
| 商品一覧画面 | 商品サービス | 商品名、価格、在庫状態を表示するため。 |
| 商品サービス | 商品一覧提供 | 注文対象になる商品を取得するため。 |
| 商品サービス | 在庫参照 | 商品ごとの在庫状態を取得するため。 |
| 在庫参照 | 在庫管理システム | REST API で在庫情報を参照するため。 |
| 注文内容確認画面 | 注文サービス | 数量指定後に注文作成を依頼するため。 |
| 注文サービス | 注文作成 | 注文の組み立てと検証を行うため。 |
| 注文作成 | 注文記録 | 組み立てた注文を保存するため。 |
| 注文記録 | リレーショナルデータベース | 注文を永続化し、注文識別子を発行するため。 |
| 注文完了画面 | 注文サービス | 作成した注文の識別子を表示するため。 |

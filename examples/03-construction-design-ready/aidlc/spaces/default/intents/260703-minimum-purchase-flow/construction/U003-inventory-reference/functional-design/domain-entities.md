# Domain Entities：在庫参照

## 目的

在庫参照に必要な Domain Entity と、商品選択および注文作成との関係を定義する。

## Domain Entity

| Entity | 責務 | 主な属性 |
|---|---|---|
| 在庫状況 | 商品ごとの在庫の有無を表す。 | 商品識別子、在庫状態、取得日時 |
| 在庫参照結果 | 在庫管理システムから取得した結果または失敗を表す。 | 商品識別子、在庫状況、取得成否、失敗理由 |
| 注文可否 | 在庫参照結果にもとづいて注文を作成できるかを表す。 | 状態、理由 |

## 関係

商品一覧提供が返す商品識別子を使って、在庫参照が在庫管理システムへ問い合わせる。
在庫参照結果は商品一覧画面の在庫状況表示と選択可否判定に使う。
注文作成は、在庫参照結果から作られた注文可否を使って注文記録の可否を判断する。

## Domain Map と Context Map への反映候補

| 候補 | 種別 | 判断 |
|---|---|---|
| 在庫参照 | 共有境界 | 未確認 |

- 在庫管理システムの責務境界は未確認である。

# Frontend Components：注文作成

## 構成

| コンポーネント | 責務 | 対応するモック |
|---|---|---|
| 注文内容表 | 選択した商品の商品名、数量、金額、合計金額を表示する | [mockups.md](../../../inception/refined-mockups/mockups.md) の注文内容確認画面 |
| 注文作成ボタン | 注文の作成を指示し、作成できたら注文完了画面へ遷移する | [mockups.md](../../../inception/refined-mockups/mockups.md) の注文内容確認画面 |
| 商品一覧へ戻るボタン | 注文を作成せずに商品一覧画面へ戻る | [mockups.md](../../../inception/refined-mockups/mockups.md) の注文内容確認画面 |
| 在庫不足の通知 | 在庫が不足しているため注文を作成できない旨を表示する | [mockups.md](../../../inception/refined-mockups/mockups.md) の「在庫が不足していた場合の表示」 |
| 在庫参照失敗の通知 | 在庫を確認できない旨と再試行の案内を表示する | [mockups.md](../../../inception/refined-mockups/mockups.md) の「在庫参照に失敗した場合の表示」 |
| 注文番号表示 | 作成した注文の注文番号と、注文番号で注文を確認できる旨を表示する | [mockups.md](../../../inception/refined-mockups/mockups.md) の注文完了画面 |

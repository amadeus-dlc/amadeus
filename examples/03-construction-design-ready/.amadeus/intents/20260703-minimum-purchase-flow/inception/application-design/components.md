# Components：最小購入フロー

greenfield の Intent であり、すべて新しいコンポーネントである。
技術スタックは未確認のため、コンポーネントは実装技術に依存しない責務の単位として定義する。

## 一覧

| コンポーネント | 責務 | 対応する要求 |
|---|---|---|
| 商品一覧画面 | 商品名、価格、在庫状況の一覧表示と、商品選択の操作の受け付け。在庫参照不能時の表示 | R001、R002、R006 |
| 注文内容確認画面 | 選択した商品の内容と合計金額の表示。注文作成と一覧へ戻る操作の受け付け。注文を作成できない場合の理由の表示 | R003、R005、R006 |
| 注文完了画面 | 注文の作成結果と注文の識別子の表示 | R004 |
| 商品カタログ | 商品情報（商品の識別子、商品名、価格）の提供 | R001 |
| 在庫参照 | 在庫管理システム（EXT001）からの在庫情報の参照。参照不能の検出と通知 | R001、R005、R006 |
| 注文 | 注文の作成、注文の識別子の発行、注文の記録と参照 | R004、R005 |

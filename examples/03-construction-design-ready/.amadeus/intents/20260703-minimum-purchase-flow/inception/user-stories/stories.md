# Stories：最小購入フロー

## 一覧

| ID | ストーリー | 対応する要求 |
|---|---|---|
| S001 | 購入者として、商品の一覧で商品名、価格、在庫状況を確認したい。なぜなら、注文する商品を選ぶ判断材料が必要だからだ。 | R001 |
| S002 | 購入者として、在庫のある商品を選んで数量を指定したい。なぜなら、欲しい商品を必要な数だけ注文したいからだ。 | R002 |
| S003 | 購入者として、注文を作成する前に注文内容と合計金額を確認したい。なぜなら、誤った内容のまま注文を作成したくないからだ。 | R003 |
| S004 | 購入者として、注文を作成して注文番号を受け取りたい。なぜなら、注文が受け付けられたことを後から確認したいからだ。 | R004 |
| S005 | 購入者として、在庫の有無を在庫管理システムの在庫情報で確認したい。なぜなら、在庫のない商品を選んで注文をやり直したくないからだ。 | R005 |
| S006 | 購入者として、在庫情報を参照できないときはそのことを知りたい。なぜなら、商品を選択できない理由が分からないと操作を続けられないからだ。 | R006 |

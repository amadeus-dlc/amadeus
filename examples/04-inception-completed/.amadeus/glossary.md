# 用語集

## 用語

| 用語 | 説明 | 状態 |
|---|---|---|
| ECサイト | 商品を公開し、購入者が商品を選択して注文できる販売チャネル。 | 確認済み |
| 商品管理 | 販売対象となる商品の情報を扱う領域。 | 確認済み |
| 顧客管理 | 購入者や顧客台帳、購入履歴を扱う領域。 | 確認済み |
| 入荷管理 | 商品の入荷を扱う領域。 | 確認済み |
| 販売管理 | 商品選択から注文作成までの販売行為を扱う領域。 | 確認済み |
| 出荷管理 | 注文後の商品出荷を扱う領域。 | 確認済み |
| 販売可能在庫 | 購入フローで販売可能かどうかを確認する在庫状態。 | 確認済み |
| 最小購入フロー | 商品選択、販売可能在庫の確認、購入者情報の記録、注文内容確認、注文作成までを扱う範囲。 | 確認済み |

# Design System Mapping：260703-minimum-purchase-flow

採用するデザインシステムは未確認である。
greenfield で UI 規約がまだ存在しないため、画面要素は汎用の UI 部品へ対応付け、部品の具体的な実装は Construction で確定する。

## 対応

| 画面要素 | デザインシステムの部品 | 備考 |
|---|---|---|
| 商品一覧の表 | テーブル | 商品名、価格、在庫状況、操作の 4 列を持つ |
| 「選択」ボタン | プライマリボタン | 在庫がある商品だけに表示する |
| 選択不可の表示 | 無効状態のテキストラベル | 操作要素を置かず、選択不可であることを文言で示す |
| 在庫情報を取得できない旨の案内 | 警告バナー | 商品一覧の表の上部に表示する |
| 数量の入力 | 数値入力フィールド | 1 以上の整数だけを受け付け、既定値は 1 にする |
| 数量の入力エラーの案内 | インラインエラーテキスト | 数量入力フィールドの近くに表示する |
| 「注文を作成する」ボタン | プライマリボタン | 数量が不正な間は無効にする |
| 「商品一覧へ戻る」ボタン | セカンダリボタン | 注文内容確認画面と注文完了画面に置く |
| 注文の識別子の表示 | 強調テキスト | 注文完了画面に表示する |

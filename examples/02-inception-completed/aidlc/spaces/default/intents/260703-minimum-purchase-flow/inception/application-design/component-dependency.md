# Component Dependency：260703-minimum-purchase-flow

## 依存関係

| 依存元 | 依存先 | 理由 |
|---|---|---|
| 商品一覧画面 | 商品サービス | 商品一覧と在庫状況の表示に使う |
| 注文内容確認画面 | 注文サービス | 注文の作成に使う |
| 注文完了画面 | なし | 注文の識別子は画面遷移で受け取り、サービスを呼ばない |
| 商品サービス | 商品一覧提供 | 商品の一覧の取得に使う |
| 商品サービス | 在庫参照 | 商品ごとの在庫状況の取得に使う |
| 注文サービス | 注文作成 | 注文の検証と組み立てに使う |
| 注文作成 | 注文記録 | 組み立てた注文の保存と識別子の発行に使う |
| 在庫参照 | 在庫管理システム（EXT001） | REST API で在庫情報を参照する |
| 注文記録 | リレーショナルデータベース | 注文の永続化に使う |

依存は上の表の一方向だけにし、循環依存を作らない。

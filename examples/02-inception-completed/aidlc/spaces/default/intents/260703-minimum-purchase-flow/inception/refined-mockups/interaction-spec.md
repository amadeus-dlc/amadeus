# Interaction Spec：260703-minimum-purchase-flow

各相互作用は、対応する要求とストーリーの識別子を参照する。

## 商品一覧の表示（R001, R006 / S001）

| 操作 | 応答 | 例外 |
|---|---|---|
| 商品一覧を開く | 在庫管理システムから在庫情報を参照し、商品名、価格、在庫状況を商品ごとに表示する | 在庫参照に失敗した場合、在庫を確認できない旨と再試行ボタンを表示し、全商品を選択不可にする（R008 / S006） |
| 再試行ボタンを押す | 在庫情報を再参照し、成功したら一覧の在庫状況を更新する | 再び失敗した場合、同じエラー表示を維持する |

## 商品選択と数量指定（R002 / S002）

| 操作 | 応答 | 例外 |
|---|---|---|
| 数量を入力する | 1 以上の整数を受け付ける。未入力の場合は 1 として扱う | 0 以下または整数でない値は受け付けず、その旨をその場で表示する |
| 在庫がある商品の選択ボタンを押す | 指定した数量で注文内容確認画面へ遷移する | 在庫がない商品は選択ボタンを無効にし、押せない |

## 注文内容の確認と注文作成（R003, R004, R007 / S003, S004, S005）

| 操作 | 応答 | 例外 |
|---|---|---|
| 注文内容確認画面を開く | 選択した商品の商品名、数量、金額、合計金額を表示する | なし |
| 商品一覧へ戻るボタンを押す | 注文を作成せずに商品一覧画面へ戻る | なし |
| 注文を作成するボタンを押す | 在庫管理システムの在庫情報を参照し、在庫が確認できた場合だけ注文を作成して注文完了画面へ遷移する | 在庫が不足している場合、注文を作成せず、その旨を表示する（R007 / S005）。在庫参照に失敗した場合、注文を作成せず、在庫を確認できない旨と再試行の案内を表示する（R008 / S006） |

## 注文完了の確認（R004, R005 / S004）

| 操作 | 応答 | 例外 |
|---|---|---|
| 注文完了画面を開く | 作成した注文の注文番号を表示する | なし |
| 商品一覧へ戻るボタンを押す | 商品一覧画面へ遷移する | なし |

# Component Dependency：最小購入フロー

## 依存関係

| 依存元 | 依存先 | 理由 |
|---|---|---|
| C001 商品一覧画面 | C004 商品閲覧サービス | 在庫状況を含む商品一覧の取得 |
| C002 注文内容確認画面 | C005 注文作成サービス | 注文作成の実行 |
| C004 商品閲覧サービス | C006 在庫参照クライアント | 在庫状況の参照 |
| C004 商品閲覧サービス | C007 商品リポジトリ | 商品情報の参照 |
| C005 注文作成サービス | C008 注文リポジトリ | 注文の記録と参照 |
| C006 在庫参照クライアント | 在庫管理システム（EXT001） | 在庫情報の REST API 参照 |
| C007 商品リポジトリ | リレーショナルデータベース | 商品情報の参照元 |
| C008 注文リポジトリ | リレーショナルデータベース | 注文の記録先 |

C003 注文完了画面は、画面遷移で注文番号を受け取るだけであり、サービスへの依存を持たない。
依存は画面からサービス、サービスから外部連携と永続化への一方向だけにする。

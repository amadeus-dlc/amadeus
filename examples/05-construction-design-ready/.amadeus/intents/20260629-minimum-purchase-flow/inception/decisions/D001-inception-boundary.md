# D001: Inception 境界判断

## 背景

- Intent の中心サブドメインは SD004 販売管理である。
- 境界づけられたコンテキストは、steering layer の既存 ID である BC004 販売管理を中心にする。
- この Intent が所有する境界づけられたコンテキストは BC004 販売管理だけにする。

## 判断

- Inception の所有境界を BC004 販売管理に固定する。
- BC004 販売管理は SD004 販売管理に対応させる。
- 商品管理、顧客管理、入荷管理、出荷管理、在庫管理は Intent 配下の境界づけられたコンテキストとして展開しない。

## 理由

- Steering layer では BC004 が商品選択から注文作成までの販売活動を扱うため。
- 最小購入フローの中心価値は販売管理にあるため。
- 依存領域を Intent 配下の境界づけられたコンテキストとして増やすと、今回の所有境界を越えるため。

## 影響

- Unit は BC004 を参照する。
- 在庫管理は SD006 として Intent 内 domain index に明示するが、BC006 は作らない。
- 商品情報と販売可能在庫の参照契約は未確認事項として残る。

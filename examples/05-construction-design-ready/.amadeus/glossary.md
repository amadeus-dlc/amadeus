# 用語集

## 用語

| 用語 | 説明 | 状態 |
|---|---|---|
| ECサイト | 商品を購入者へ販売するためのサイト。 | 確認済み |
| 商品管理 | 商品情報を管理し、販売対象の商品を扱う領域。 | 確認済み |
| 顧客管理 | 顧客情報、会員登録、ログイン、顧客台帳、購入履歴を扱い得る領域。 | 確認済み |
| 入荷管理 | 商品の入荷を扱う領域。 | 確認済み |
| 販売管理 | 商品選択から注文作成までの購入フローを扱う領域。 | 確認済み |
| 出荷管理 | 注文後の商品出荷を扱う領域。 | 確認済み |
| 販売可能在庫 | 購入フローで販売できるかを確認する在庫。 | 確認済み |
| 注文 | 購入者情報と商品選択をもとに作成される購入申込の記録。 | 確認済み |

## 避ける語

| 避ける語 | 代わりに使う語 | 理由 |
|---|---|---|
| EC全体 | ECサイト構築 | 対象範囲が広すぎるため。 |
| 在庫確保 | 在庫引当 | 後続候補の範囲と用語を合わせるため。 |

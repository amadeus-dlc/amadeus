## 設計

秘密情報を扱わないため、専用のセキュリティ機構は設けない。遡及補完は公開情報の転記に限る（nfr-requirements の security 判断のとおり）。

## 判断の位置づけ

nfr-requirements の要求を設計へ写像したものであり、新しい機構は追加しない（暫定機構 C07）。実装の一次情報は functional-design と application-design の契約である。

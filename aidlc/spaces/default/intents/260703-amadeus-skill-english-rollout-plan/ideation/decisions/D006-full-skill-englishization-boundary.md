# D006：全面英語化の完了境界

## 判断

Issue #399 は、#395、#400、#401、#402 の完了だけでは閉じない。

Amadeus 系 `SKILL.md` の全面英語化が完了するまで、この Intent を継続する。

## 状態

active

## 根拠

Issue #399 は、推奨方針「方針確定 → 小さい英語化 PR → 意味差分対応 → 残り skill の段階的英語化」を親タスクとして管理する。

#402 は残り skill の段階的英語化単位を決める Issue であり、実際の全面英語化完了を代替しない。

## 影響

- PR #414 の merge は、直接子 Issue の計画・追跡完了として扱う。
- PR #416 の完了記録は取り下げ、Issue #399 を閉じない。
- #391〜#394 の差分対応と RU002〜RU006 の段階的英語化を、Issue #399 の残タスクとして追跡する。

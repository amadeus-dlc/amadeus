# Memory：units-generation

## 解釈

Unit 境界は、子 Issue 単位を基本にした機能別、中粒度として扱う。

#395、#400、#401、#402 を Unit として定義する。

#401 配下の #391、#392、#393、#394 は、#401 Unit に内包する。

## 逸脱

Bolt、実施順序、PR 作成手順、実装コードは作成しない。

実施順序と Bolt 分割は Delivery Planning に委ねる。

## トレードオフ

子 Issue 単位にすると、GitHub Issue の完了証拠と Unit の境界が一致する。

一方で、要求単位よりは粒度が粗くなるため、R004 の PR 境界確認は各 Unit の横断観点として扱う。

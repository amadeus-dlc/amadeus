# Design System Mapping：Amadeus skill 英語化実施計画

## 概要

この Intent は UI を対象にしない。

そのため、デザインシステムのコンポーネント割り当ては該当なしとする。

## 表現要素

| 要素 | 表現 | 目的 |
|---|---|---|
| 子 Issue | Mermaid node | 順序と依存関係を確認する。 |
| 完了証拠 | Mermaid node | PR merge または Issue close を確認する。 |
| 相互作用 | Mermaid edge | 成果物、GitHub、判断の流れを確認する。 |
| 対応要求 | Markdown の参照 ID | 要求と相互作用を追跡する。 |
| 対応ストーリー | Markdown の参照 ID | アクター価値と相互作用を追跡する。 |

## 非採用

画面レイアウト、フォーム、ボタン、色、余白、タイポグラフィは定義しない。

この stage では実装対象の UI を作らないためである。
